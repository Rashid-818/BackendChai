# 🚀 Backend Project Setup

Yeh mera backend project hai jisme maine modular architecture (MVC) aur professional file structure follow kiya hai.


# 📂 Project Architecture

Maine project ko clean aur scalable rakhne ke liye organize kiya hai:

controllers/: Request handlers (Business logic).

db/: Database configuration and connection logic.

middlewares/: Custom validation, authentication, and error handling.

models/: Database schema definitions (MongoDB models).

routes/: API endpoint definitions.

utils/: Reusable helper functions.

.env: Environment variables for sensitive configuration (DB URI, PORT).

# 🛠 Features

Modular Structure: Easy to maintain and scale.

Database Connection: Integrated with MongoDB/Mongoose.

Environment Config: Used .env for secure configuration.

# ⚙️ How to Setup

Prerequisites
   Node.js (v14 or higher)
   MongoDB
